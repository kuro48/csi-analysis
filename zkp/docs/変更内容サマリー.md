## 🎓 技術的なポイント

### 数学的アプローチ
```
コサイン類似度 = (A·B) / (||A|| × ||B||)

固定小数点変換:
  実数 0.85 → 8500 (×10^4)
  実数 0.62 → 6200 (×10^4)

計算例:
  similarity = (138080000 × 10000) / (11747 × 11831)
             = 9990 ≈ 0.999
```

### Witness検証メカニズム
```circom
// JavaScript側で計算
const normA = Math.sqrt(Σ Ai²);
const similarity = (A·B × SCALE) / (normA × normB);

// Circom側で検証
lhs <== similarity × normA × normB;
rhs <== dotProduct × SCALE;
lhs === rhs;  // 制約
```

### プライバシー保護
- **公開**: 選択インデックス、類似度、ZKP証明
- **秘匿**: 生CSIデータ、中間計算値
- **検証可能**: 選択の正当性、計算の正確性

## 🔧 使い方

### クイックスタート
```bash
# 1. テスト実行
cd zkp
node scripts/test_csi_selector.js

# 出力例:
# ===== テストケース4: サブキャリア選択 =====
# 候補1の類似度: 0.998954
# 候補2の類似度: 0.828053
# 選択されるべき候補: 候補1
# 全テスト完了 ✓

# 2. 回路コンパイル
node scripts/compile_csi_selector.js

# 3. 証明生成・検証
# (詳細は README_CSI_Integration.md を参照)
```

### パラメータ設定
```circom
// circuits/csi_subcarrier_selector.circom の最終行

component main {public [reference, candidates]} =
  CSISubcarrierSelector(4, 2, 10000, 7000);
//                      │  │    │      │
//                      │  │    │      └─ 閾値 (0.7)
//                      │  │    └──────── スケール (10^4)
//                      │  └─────────── 候補数 (2個)
//                      └────────────── 次元数 (4次元)
```
