# ドキュメント

このディレクトリには、CSI呼吸監視システムに関する各種ドキュメントが含まれています。

## ディレクトリ構造

### api/
API仕様とエンドポイント情報
- `API_ENDPOINTS.md` - APIエンドポイント一覧と使用方法

### backend/
バックエンドアーキテクチャと設計
- `BACKEND_SPECIFICATION.md` - バックエンドの詳細仕様
- `backend-architecture.md` - アーキテクチャ設計

### deployment/
デプロイメント関連
- `DEPLOYMENT.md` - デプロイ手順とベストプラクティス

### reports/
分析レポートとコードレビュー
- `cleanup-report.md` - コードクリーンアップレポート
- `code-analysis-report.md` - コード分析レポート
- `comprehensive_code_analysis_report.md` - 包括的コード分析
- `improvement-phase1-report.md` - 改善フェーズ1レポート
- `improvement-phase2-partial-report.md` - 改善フェーズ2レポート

### research/
研究資料と参考文献
